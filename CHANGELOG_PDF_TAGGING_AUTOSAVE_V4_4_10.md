# IARS v4.4.10 — PDF Tagging Automatic Save

- Retains the approved single-header layout for Generate Extraction, PDF Tagging, and Shared PDF Archive.
- Adds per-textbox font size control from 6–48 pt.
- Preserves the selected font size after deselection and reselection.
- New textboxes inherit the latest chosen font size.
- Removes the manual Save Changes button.
- Automatically saves textbox text, position, size, style, and font size after a short pause or control change.
- Keeps editing local while actively typing to reduce Streamlit reruns and lag.
- Displays Editing / Saving / All changes saved automatically status messages.
- Uses the saved font size in generated tagged PDFs.
