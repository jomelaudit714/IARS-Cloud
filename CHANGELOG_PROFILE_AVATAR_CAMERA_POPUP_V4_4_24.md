# IARS V4.4.24 – Camera Button on User Card + Simplified Popup Editor

## Updated profile picture flow
- Removed the old **Change Profile Picture** expander.
- Added a visible **camera badge** on the top-right user card avatar.
- The profile picture editor is now shown directly inside the profile popup instead of a separate expander section.
- The editor now uses only:
  - **zoom in / zoom out** controls
  - **drag-based cropping**
- Removed the previous left/right and up/down manual position sliders from the profile picture editor flow.

## Retained
- Live circular preview before saving.
- Save / remove profile picture actions.
- Optimized 320 x 320 JPEG output.
