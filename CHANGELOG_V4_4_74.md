# Changelog — V4.4.74

## Weekly Itinerary preview

- Changed the Weekly Itinerary popup from large to medium width.
- Added aspect-ratio-aware sizing so the full image remains inside the screen.
- The supplied sample itinerary renders at approximately 589 × 433 pixels in the popup.
- The original image remains downloadable.

## Dashboard

- Moved Weekly Itinerary to the left side of the Dashboard.
- Moved Recent Archive Activity to the right side.
- Approved auditor itineraries now appear automatically on the Dashboard.
- Removed the extra Dashboard `View Itinerary` button.
- Retained the direct download button.
- The supplied sample itinerary renders at approximately 794 × 583 pixels in the main Dashboard area.

## Metric cards

- Dashboard metric cards now use five equal horizontal columns instead of reserving an unused sixth column.
- Increased card height, padding, and value text size.
- Responsive three-column and two-column layouts remain for smaller screens.

## Unchanged

- Weekly Itinerary upload, privacy, approval, return, and revision workflow
- Generate Extraction and Operations Audit rules
- PDF Tagging
- Duplicate notification
- Policies & Memoranda folders
- authentication, avatar, archive, and Master Data modules
