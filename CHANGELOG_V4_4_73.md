# Changelog — V4.4.73

## Weekly Itinerary module

- Added a new `Weekly Itinerary` sidebar module for all signed-in auditors.
- Auditors may upload one JPG, JPEG or PNG itinerary image per week.
- Each auditor can view only their own submissions.
- Administrators can view all auditor submissions, filter them, open the image, download it, approve it, or return it for revision.
- Approved itineraries appear on the submitting auditor's Dashboard.
- Pending and returned statuses are shown to the auditor.
- Returned submissions may be revised and resubmitted while retaining prior revisions for audit history.
- Maximum upload size is 10 MB.
- Images are stored in a private Supabase bucket.
