# Changelog — V4.4.75

## Dashboard vertical spacing

- Raised the Dashboard content to recover unused space above the header.
- Reduced spacing between the application header, Welcome Back section, and metric cards.
- Kept the layout responsive on smaller screens.

## Sidebar alignment

- Raised the sidebar content closer to the EDL logo.
- Forced the EDL logo and its image wrappers to align horizontally at the center.
- Reduced unnecessary spacing below the brand and before the navigation controls.

## Metric cards

- Retained five equal cards across the full Dashboard width.
- Increased card-label, value, note, and icon sizes.
- Vertically centered the card text so the larger boxes are properly utilized.
- Retained three-column and two-column responsive layouts.

## Weekly Itinerary Dashboard behavior

- The signed-in user's approved itinerary for the current calendar week is displayed automatically as JPG/PNG.
- The image stretches to the available left Dashboard column while preserving its aspect ratio.
- Ordinary auditors do not need to press a View button.
- Administrators still see the pending-approval summary first.
- An administrator's own approved current itinerary is displayed below the pending summary.
- Other auditors' itinerary images are never displayed in the administrator's personal Dashboard panel.
- Recent Archive Activity remains on the right side.
