# Changelog — IARS v4.1.0

## Interface

- Rebuilt the application theme to more closely follow the approved EDL enterprise mockup.
- Preserved the exact original EDL GROUP OF COMPANIES logo file.
- Reworked the login page into a professional navy visual panel and white account-access panel.
- Replaced the large repeated application banner with a compact page-specific top bar.
- Refined sidebar branding, navigation, user card, and system status.
- Rebuilt Dashboard with six KPI cards, recent archive activity, vertical quick actions, and system overview.
- Added workflow steppers to Generate Extraction and PDF Tagging.
- Added Shared PDF Archive summary metrics.
- Refined forms, tabs, tables, file uploaders, alerts, cards, borders, typography, spacing, and responsive behavior.
- Changed trusted local HTML rendering to `st.html()` where available to prevent raw HTML from appearing.

## Functional scope

No extraction, authentication, storage, compression, tagging, document-library, Master Data, or export business logic was intentionally removed.
