# IARS V4.4.23 – Avatar Camera Icon + Crop Flow

## Updated profile picture flow
- Added a compact camera-icon upload trigger for the profile picture section.
- The upload control is now styled as a camera icon instead of a large default uploader panel.
- Added zoom controls with minus and plus buttons.
- Added a `Crop photo (drag the image)` toggle.
- When crop mode is enabled and the cropper component is available, the user can drag and crop the picture interactively.
- The live preview card still shows the expected circular avatar result before saving.

## Retained
- Output is still optimized to a 320x320 JPEG.
- Save / remove profile picture functions remain connected to the existing storage/profile flow.
