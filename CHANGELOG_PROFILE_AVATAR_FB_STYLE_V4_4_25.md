# IARS V4.4.25 – Avatar Menu (See Avatar / Change Avatar)

## Updated avatar behavior
- Clicking the top-right user card opens the profile popup.
- Inside the popup, the avatar section now shows only two main actions:
  - **See Avatar**
  - **Change Avatar**
- **See Avatar** displays a larger view of the current avatar.
- **Change Avatar** opens the upload/edit flow.
- The edit flow now uses only:
  - **plus / minus zoom controls**
  - **drag-based crop positioning**
  - **Save** and **Cancel** buttons
- Removed the old left/right and up/down manual movement sliders.
- Removed the old Change Profile Picture expander flow.

## Retained
- Camera badge on the user-card avatar.
- Live preview card during avatar editing.
- Optimized 320x320 saved avatar output.
