# Changelog — V4.4.80

## Policies & Memoranda preview

- The View icon now opens a separate centered medium-sized popup.
- The popup remains open after the View action and no longer returns immediately to the folder cards.
- Closing the preview or selecting Back to Folder returns to the same company folder.
- PDF, DOCX, and XLSX previews are constrained to the screen for readability.
- Original document download remains available inside the preview.

## Administrator controls

- Administrators can edit document title, type, subject/process category, description, version, and effective date.
- Administrators can permanently delete an uploaded policy or memorandum after confirmation.
- Editing updates metadata only and does not replace the stored file.

## Login artwork

- The login left-panel artwork is visually shifted slightly right so the EDL logo aligns horizontally with the Internal Audit Report System title.

## Unchanged

- Company folders, title search, row View/Download icons, and subject/process categories
- Generate Extraction and all audit parsing rules
- PDF Tagging, Weekly Itinerary, Dashboard, archive, authentication, and Master Data
